Taschenrechner (Python-Projekt)

Dies ist ein einfacher Taschenrechner, der die vier Grundrechenarten unterstützt: Addition, Subtraktion, Multiplikation und Division.  
Das Programm prüft die Benutzereingaben auf Fehler und gibt hilfreiche Hinweise bei falscher Eingabe.

Funktionen

- Addieren zweier Zahlen
- Subtrahieren zweier Zahlen
- Multiplizieren zweier Zahlen
- Dividieren zweier Zahlen mit Fehlerbehandlung bei Division durch Null
- Fehlerprüfung für falsche Eingaben (z. B. Buchstaben oder Symbole)
- Unterstützung für Dezimalzahlen (z. B. 4.5)

Bedienung

1. Wähle eine Rechenart aus: 1, 2, 3 oder 4
2. Gib zwei Zahlen ein (z. B. 3.5 und 2)
3. Das Ergebnis wird angezeigt
